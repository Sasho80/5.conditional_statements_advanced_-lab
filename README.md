01.Problem: Day of the week
Write a program that reads an integer entered by the user and prints the day of the week (in English), 
in the range [1...7], or prints "Error" in case the entered number is invalid.

input	output
1	    Monday
2	    Tuesday
3   	Wednesday
4	    Thursday
5	    Friday
6	    Saturday
7	    Sunday
-1	  Error
